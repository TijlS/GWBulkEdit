# Security Policy

## Supported Versions

Previous versions will (probably) still work, but aren't supported anymore by us.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.0   | ❎ |
| 2.0.x   | ❎ |
| 3.x.x   | ✅ |


## Reporting a Vulnerability

If you find a security vulnerability, please create an issue for it.
